Wizard
======
A cross-platform TCP server based on .Net Framework 4.5.

Features
--------
* Using IoC for component management.
* Support Multi potocols, e.g. Plain string, protobuf.
* Build on the top of TPL and async-await.
* Support .Net/Mono.

Deps
----
Managed by NuGet
* Castle Windsor (For IoC)
* Couchbase (Storage)
* IronPython
* log4net
* protobuf (optional)
